git flow ( Master / Develop / Feature / Hotfix)

# How to use git flow

```shell

1. $ git flow init
2. ENTER ENTER ENTER ...
3. Then you can see the develop branch ( $ git branch )
4. $ git flow feature start ( feature branch name ) 
5. Then you can see the feature branch ( $ git branch )
6. Then make file and work
7. add / commit
8. If you finish the work, $ git flow feature finish (file name)
9. If you want release, git flow release start ( version name ex:v0.0.0 )
10. git flow release finish ( version name )
11. Then you can see 3 vim ( main / tag / develop )
12. $ git push -u origin develop
13. $ git checkout main
14. $ git push origin main
15. $ git tag
16. $ git push --tag

```
